安装hexo
  1. 在电脑打开git
    cnpm install -g hexo-cli
  2. 在电脑里面建立自己的文件夹 blog然后在文件夹里打开git
    输入 hexo init 初始化 hexo
  3. 安装一些依赖
    hexo install
  4. 安装一下插件
    cnpm install hexo-deployer-git –save
  5. 打开github新建一个项目仓库的名字和你的用户名一样github.io
  6. 复制你的地
  7. 找到_config.ym文件修改成这样

deploy:
type: git
repository: https://github.com/xiejinjun1217/xiejinjun1217.github.io.git
branch: master

  1. 把hexo 博客部署到github上面
    hexo g -d
  2. 输入你的主页比如我的
    xiejinjun1217.github.io.git
